# Expense Tracker

A Java-based application that helps users track and manage their daily expenses efficiently.
This project demonstrates basic backend concepts such as expense management, user handling, and database connectivity.


## Features:

- Add and manage daily expenses
- User management functionality
- Database connectivity using JDBC
- Organized backend structure using Java classes
  

## Technologies Used:

- Java
- JDBC (Java Database Connectivity)
- MySQL Database
  

## Project Structure:

Expense-Tracker
│
├── DatabaseConnection.java   # Handles database connection
├── ExpenseManager.java       # Contains expense management logic
├── ExpenseTrackerApp.java    # Main application entry point
└── UserManager.java          # Handles user-related operations


## How to Run the Project:

1. Clone the repository.
2. Import the project into your Java IDE (NetBeans / IntelliJ).
3. Configure the MySQL database connection in "DatabaseConnection.java".
4. Run "ExpenseTrackerApp.java".
